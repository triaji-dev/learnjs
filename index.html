The learnJS Module is a foundational JavaScript guide that emphasizes practical application within modern web development frameworks. Utilizing the Just-in-Time Learning (JITL) method, the learning focus is on step-by-step coding exercises to achieve a thorough understanding and train the participants' muscle memory.

Methods:

  • Learning JavaScript more targeted and focused, aligned with its relevance and use within the modern React framework.
  • Using the Just-in-Time Learning (JITL) method, JavaScript material is delivered incrementally according to the micro-projects being worked on.
  • Participants engage in hands-on practice gradually; the main goal is to achieve both comprehension and train muscle memory in the use of syntax.

Fungsi Fundamental Frontend Web Developer

According to the diagram above, what a Frontend Web Developer fundamentally does, in simple terms, is:

  • Managing data using an API (from the Backend), which includes CRUD (Create, Read, Update, Delete) operations.
  • Managing the overall state (Component Local State, Global Application State, User Event State, Server State).
  • Converting the UI/UX design from Figma (from the Designer) into code, which involves the DOM, design tokens, UI implementation, and assets.

Therefore, the JavaScript learning module should broadly focus on mastering material that is relevant to these three areas within the React ecosystem.

Module Description

  • This module focuses on mastering JavaScript material that is relevant to the three main pillars of the React ecosystem: data management, state management, and UI/UX management.
  • Learning utilizes the Just-in-Time Learning (JITL) method. The material is explained incrementally according to relevant examples of its usage within the React ecosystem.
  • Learning through the "Building Blocks" approach (constructing piece by piece towards a single final goal) is the best way to understand the React ecosystem.
  • Throughout this module, mentees will build a Final Project named: "TaskMaster Dashboard" (a simple task management application featuring Create, Read, Update, Delete, Filtering, and Statistics).
