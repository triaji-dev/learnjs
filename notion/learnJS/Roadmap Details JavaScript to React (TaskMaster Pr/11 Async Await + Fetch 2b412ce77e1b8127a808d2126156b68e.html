11. Async/Await + Fetch (15 Menit)

A. Konsep Dasar Asynchronous Programming

Asynchronous Programming adalah kunci dalam pengembangan web. Ketika aplikasi Anda meminta data dari server (misalnya, memanggil API), operasi ini membutuhkan waktu, yang sering disebut Latency.

Jika JavaScript menunggu (bersifat synchronous) sampai data diterima, seluruh aplikasi akan hang (macet) dan tidak responsif.

Asynchronous berarti kode berjalan di background (misalnya, mengambil data) tanpa memblokir thread utama, sehingga UI tetap responsif.

Promise

Secara historis, operasi asinkron ditangani dengan Callback (yang dapat menyebabkan Callback Hell). Saat ini, standar modern adalah Promise.

  • Promise adalah objek yang mewakili penyelesaian atau kegagalan asinkron di masa depan.
  • Sebuah Promise dapat berada dalam salah satu dari tiga status:
    • Pending: Operasi sedang berjalan (belum selesai).
    • Fulfilled (Resolved): Operasi selesai dengan sukses.
    • Rejected: Operasi gagal.

B. Fetch API dan Async/Await

fetch() adalah API modern berbasis Promise di browser yang digunakan untuk membuat permintaan jaringan (HTTP) ke server.

1. fetch() (Menggunakan .then() secara Tradisional)

JavaScript

fetch('https://jsonplaceholder.typicode.com/todos?_limit=5')
    .then(response => {
        // Response pertama adalah objek Response HTTP. 
        // Kita harus memanggil .json() untuk menguraikan body-nya.
        return response.json(); 
    })
    .then(data => {
        // Data adalah objek/array JavaScript yang sudah diuraikan
        console.log(data);
    })
    .catch(error => {
        // Menangani kegagalan jaringan atau kesalahan lainnya
        console.error('Error fetching data:', error);
    });

2. async/await (Sintaks Modern)

async/await adalah sintaks ES2017 yang memungkinkan Anda menulis kode asinkron yang terlihat dan terasa seperti kode synchronous biasa, membuatnya jauh lebih mudah dibaca.

  • async: Kata kunci yang harus diletakkan sebelum deklarasi fungsi. Fungsi async selalu mengembalikan sebuah Promise.
  • await: Kata kunci yang hanya dapat digunakan di dalam fungsi async. Ia akan menghentikan eksekusi fungsi async sampai Promise di sebelahnya berhasil diselesaikan (resolved), dan kemudian mengembalikan nilai yang diselesaikan tersebut.

Contoh Menggunakan async/await:

JavaScript

async function fetchTasks() {
    try {
        // 1. Tunggu (await) respons dari fetch
        const response = await fetch('https://jsonplaceholder.typicode.com/todos?_limit=5');
        
        // Cek apakah respons OK (status 200-299)
        if (!response.ok) {
            throw new Error(`HTTP error! status: ${response.status}`);
        }
        
        // 2. Tunggu (await) penguraian body (response.json())
        const data = await response.json(); 
        
        // 3. Gunakan data
        console.log(data);
        return data;

    } catch (error) {
        // Tangani error (baik dari fetch maupun jika response.ok gagal)
        console.error('Gagal mengambil data tugas:', error);
        return [];
    }
}

C. Relevansi React: useEffect & API Calls

Dalam React, side effects (seperti pengambilan data dari API) harus dilakukan di dalam Hook useEffect.

  • Ketika Anda ingin memuat data saat komponen pertama kali dimuat (mounted), Anda akan menggunakan useEffect dengan dependency array kosong ([]).
  • Fungsi yang Anda oper ke useEffect tidak boleh berupa async secara langsung. Solusi terbaik adalah membuat fungsi async terpisah di dalamnya dan langsung memanggilnya.

Contoh Pseudo-Code React:

JavaScript

import React, { useState, useEffect } from 'react';

function TaskList() {
    const [tasks, setTasks] = useState([]);
    const [isLoading, setIsLoading] = useState(true);

    useEffect(() => {
        // Buat fungsi async di dalam useEffect
        async function loadTasks() {
            try {
                const response = await fetch('...');
                const data = await response.json();
                setTasks(data); // Perbarui state dengan data API
            } catch (error) {
                console.error(error);
            } finally {
                setIsLoading(false);
            }
        }
        loadTasks(); // Panggil fungsi async
        
        // Dependency array kosong: Hanya berjalan saat mount
    }, []); 

    if (isLoading) return <p>Memuat Tugas...</p>;

    // ... Render tasks ...
}

Kita akan mengganti data dummy TaskMaster dengan data nyata dari JSONPlaceholder menggunakan async/await dan fetch().

https://jsonplaceholder.typicode.com/todos?_limit=10 (Mengambil 10 dummy tugas/todo).

Exercise untuk peserta:

  1. Buat struktur HTML dengan area untuk status loading dan daftar tugas.
  1. Tulis fungsi showLoading() dan hideLoading() untuk feedback UX.
  1. Tulis fungsi fetchAndRenderTasks() yang dideklarasikan sebagai async.
  1. Di dalam fungsi tersebut, gunakan await fetch() dan await response.json().
  1. Petakan data yang diterima ke struktur yang Anda butuhkan (misalnya, title dan completed status).
  1. Tampilkan data yang sudah dipetakan ke dalam DOM.

Contoh Code Starter (HTML/JS):

HTML

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Praktek: Async/Await + Fetch</title>
    <script src="https://cdn.tailwindcss.com"></script>
</head>
<body class="bg-gray-100 p-8">

    <div id="app" class="max-w-xl mx-auto bg-white p-6 rounded-lg shadow-xl">
        <h1 class="text-3xl font-bold mb-6 text-gray-800">TaskMaster: Data API</h1>
        
        <p id="loading-status" class="text-center text-indigo-500 font-semibold mb-4">Memuat data dari server...</p>
        
        <ul id="task-list" class="space-y-3">
            </ul>
    </div>

    <script>
        const API_URL = 'https://jsonplaceholder.typicode.com/todos?_limit=10';
        const taskListContainer = document.getElementById('task-list');
        const loadingStatus = document.getElementById('loading-status');

        function showLoading() {
            loadingStatus.classList.remove('hidden');
            taskListContainer.innerHTML = '';
        }

        function hideLoading() {
            loadingStatus.classList.add('hidden');
        }

        // --- FUNGSI ASINKRON UTAMA ---
        async function fetchAndRenderTasks() {
            showLoading(); // Tampilkan status loading
            
            try {
                // 1. Gunakan 'await' untuk menunggu respons fetch (Promise)
                const response = await fetch(API_URL);

                if (!response.ok) {
                    throw new Error(`Gagal mengambil data (Status: ${response.status})`);
                }

                // 2. Gunakan 'await' untuk menunggu penguraian JSON (Promise)
                const data = await response.json(); 
                
                // 3. Memetakan dan merender data ke UI
                const taskItems = data.map(todo => {
                    // JSONPlaceholder menggunakan 'completed' (boolean), kita ubah ke label
                    const statusText = todo.completed ? 'Selesai' : 'Pending';
                    const statusClass = todo.completed ? 'bg-green-200 text-green-800' : 'bg-yellow-200 text-yellow-800';
                    
                    return `
                        <li class="flex justify-between items-center p-3 bg-gray-50 rounded-lg border">
                            <span class="text-lg font-medium ${todo.completed ? 'line-through text-gray-500' : 'text-gray-800'}">${todo.title}</span>
                            <span class="px-2 py-1 text-xs font-semibold rounded-full ${statusClass}">${statusText}</span>
                        </li>
                    `;
                }).join(''); // Menggabungkan array string menjadi satu string HTML

                taskListContainer.innerHTML = taskItems;

            } catch (error) {
                // Tangani error dan tampilkan pesan
                taskListContainer.innerHTML = `<li class="p-4 text-red-600 font-semibold">Error: ${error.message}</li>`;
                console.error("Kesalahan Asinkron:", error);
            } finally {
                // Selalu sembunyikan status loading, baik sukses maupun gagal
                hideLoading(); 
            }
        }

        // --- INISIALISASI ---
        document.addEventListener('DOMContentLoaded', fetchAndRenderTasks);
    </script>
</body>
</html>
