15. Value vs Reference (15 Menit)

A. Konsep Dasar Data Primitif (Pass by Value)

Dalam JavaScript, terdapat dua kategori besar tipe data: Primitif dan Non-Primitif (Objek). Cara penyimpanan dan perbandingan keduanya sangat berbeda.

Tipe Data Primitif (Value)

Tipe data: String, Number, Boolean, null, undefined, Symbol, BigInt.

Ketika Anda bekerja dengan data primitif, nilai variabel disimpan secara langsung di tempatnya. Ketika Anda menyalin atau membandingkannya, yang diperhatikan adalah nilai aktual di dalamnya.

JavaScript

let a = 10;
let b = 10;

console.log(a === b); // Hasil: true (Karena 10 sama dengan 10)

let x = 'Hello';
let y = x; // Variabel y mendapatkan salinan nilai 'Hello'
x = 'World';

console.log(y); // Hasil: 'Hello' (y tidak berubah)

Perbandingan === pada primitif akan mengembalikan true jika nilainya sama.

B. Konsep Data Non-Primitif (Pass by Reference)

Tipe Data Non-Primitif (Reference)

Tipe data: Object, Array, Function, Date.

Ketika Anda bekerja dengan data non-primitif, nilai variabel tidak menyimpan data itu sendiri, melainkan menyimpan referensi (alamat memori) ke lokasi di mana data tersebut disimpan di dalam memori komputer.

Perbandingan Referensi

Perbandingan === pada objek hanya akan mengembalikan true jika kedua variabel menunjuk ke alamat memori yang persis sama.

1. Perbandingan Dua Objek yang Sama Secara Nilai:

JavaScript

const obj1 = { name: 'A' };
const obj2 = { name: 'A' };

// Walaupun isinya sama, mereka adalah dua objek berbeda 
// yang disimpan di dua alamat memori berbeda.
console.log(obj1 === obj2); // Hasil: false ❌

2. Perbandingan Array Kosong:

Ini menjawab pertanyaan penting: [] === [] adalah false.

Setiap kali Anda menggunakan notasi literal ([] atau {}), JavaScript membuat objek baru di lokasi memori baru.

JavaScript

console.log([] === []);   // Hasil: false
console.log({} === {});   // Hasil: false

Menyalin Referensi

Ketika Anda menyalin objek tanpa kloning mendalam, Anda hanya menyalin alamat memori. Kedua variabel akan menunjuk ke data yang sama.

JavaScript

const originalTask = { title: 'Laporan', status: 'Pending' };
const sameTask = originalTask; // Menyalin REFERENSI (alamat)

sameTask.status = 'Completed'; // Mengubah melalui sameTask

console.log(originalTask.status); // Hasil: 'Completed' (Keduanya berubah!)
console.log(originalTask === sameTask); // Hasil: true (Mereka menunjuk ke lokasi yang sama)

C. Fitur TaskMaster: Cek Duplikasi (Deep Equality)

Dalam TaskMaster, kita perlu mencegah pengguna memasukkan tugas dengan judul yang sama.

Kita tidak bisa membandingkan seluruh objek tugas (taskA === taskB) karena hasilnya akan selalu false. Kita perlu membandingkan nilai di dalam objek. Ini disebut Deep Equality (kesamaan mendalam).

Implementasi Cek Duplikasi Judul:

Kita harus menggunakan .some() atau .filter() untuk mengiterasi array tugas dan membandingkan properti primitif (judul, yang berupa string) dari setiap elemen.

JavaScript

const existingTasks = [
    { id: 1, title: 'Input Data', isCompleted: false },
    { id: 2, title: 'Review Code', isCompleted: true }
];

const newTaskTitle = 'Input Data'; 

// Cek apakah ada tugas yang title-nya SAMA
const isDuplicate = existingTasks.some(existingTask => {
    // Membandingkan dua string (Value Comparison)
    return existingTask.title === newTaskTitle; 
});

console.log(isDuplicate); // Hasil: true

D. Relevansi React: Dependency Array useEffect

Pemahaman tentang Value vs Reference sangat penting di React, terutama saat menggunakan Hooks seperti useEffect.

Jika Anda meletakkan objek atau array di dalam Dependency Array ([]) pada useEffect, Hook tersebut akan berjalan pada setiap render, bahkan jika isinya tidak berubah, karena React membandingkan objek tersebut berdasarkan referensi.

JavaScript

const userOptions = { sort: 'asc', limit: 10 };

useEffect(() => {
    console.log("Effect berjalan!");
    // ... logic API call ...
}, [userOptions]); // userOptions adalah objek (Reference Type)

// Di setiap render, userOptions akan menjadi objek baru di memori baru
// Walaupun isinya { sort: 'asc', limit: 10 } sama, REFERENSI-nya beda!
// ✅ Solusi: Jika memungkinkan, pisahkan objek menjadi primitif (misal: [userOptions.sort, userOptions.limit])

Live Coding: Cek Duplikasi Tugas (15 Menit)

Kita akan memodifikasi form TaskMaster untuk mencegah penambahan tugas jika judulnya sudah ada, menggunakan konsep Value Comparison di dalam array.

Exercise untuk peserta:

  1. Buat array tugas awal yang berisi beberapa tugas.
  1. Tulis fungsi isDuplicate(title) yang menggunakan .some() untuk membandingkan judul tugas.
  1. Modifikasi event handler addTask agar:
    • Mencegah default action jika input kosong.
    • Mencegah default action jika isDuplicate(title) mengembalikan true.
    • Memberikan feedback yang sesuai kepada pengguna.

Contoh Code Starter (HTML/JS):

HTML

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Praktek: Value vs Reference & Duplikasi</title>
    <script src="https://cdn.tailwindcss.com"></script>
</head>
<body class="bg-gray-100 p-8">

    <div id="app" class="max-w-xl mx-auto bg-white p-6 rounded-lg shadow-xl">
        <h1 class="text-3xl font-bold mb-6 text-gray-800">Cek Duplikasi Tugas</h1>
        
        <p id="feedback" class="mb-4 p-2 rounded text-sm text-center hidden"></p>

        <form id="task-form" class="mb-6 pb-4 border-b flex space-x-2">
            <input type="text" id="new-task-title" placeholder="Tambahkan tugas baru..." 
                   class="flex-grow p-2 border border-gray-300 rounded-md focus:outline-none focus:ring-2 focus:ring-indigo-500">
            <button type="submit" id="add-task-btn" class="bg-indigo-600 text-white p-2 rounded-md hover:bg-indigo-700">
                Tambah
            </button>
        </form>

        <h2 class="text-xl font-semibold mb-3 text-gray-800">Daftar Tugas (${tasks.length})</h2>
        <ul id="task-list" class="space-y-3">
            </ul>
    </div>

    <script>
        // --- STATE AWAL ---
        let tasks = [
            { id: 1, title: 'Belajar Value vs Reference', isCompleted: false },
            { id: 2, title: 'Buat Demo Cek Duplikasi', isCompleted: true },
        ];

        const taskListContainer = document.getElementById('task-list');
        const newTaskTitleInput = document.getElementById('new-task-title');
        const feedbackElement = document.getElementById('feedback');

        // --- FUNGSI UTAMA: CEK DUPLIKASI ---
        function isDuplicate(newTitle) {
            const normalizedNewTitle = newTitle.trim().toLowerCase();
            
            // ✅ Menggunakan .some() untuk Value Comparison pada properti 'title' (String)
            return tasks.some(task => {
                const normalizedExistingTitle = task.title.trim().toLowerCase();
                return normalizedExistingTitle === normalizedNewTitle; 
            });
        }

        // --- MANIPULASI UI ---
        function renderTasks() {
            taskListContainer.innerHTML = '';
            document.querySelector('h2').textContent = `Daftar Tugas (${tasks.length})`;
            
            tasks.forEach(task => {
                const li = document.createElement('li');
                li.className = 'p-3 bg-gray-50 rounded-lg border';
                li.textContent = task.title;
                taskListContainer.appendChild(li);
            });
        }
        
        function updateFeedback(message, type = 'success') {
            feedbackElement.textContent = message;
            feedbackElement.classList.remove('hidden', 'bg-red-100', 'text-red-700', 'bg-green-100', 'text-green-700');
            
            if (type === 'error') {
                feedbackElement.classList.add('bg-red-100', 'text-red-700');
            } else {
                feedbackElement.classList.add('bg-green-100', 'text-green-700');
            }
        }

        // --- HANDLE SUBMIT ---
        function handleAddTask(e) {
            e.preventDefault();
            const title = newTaskTitleInput.value.trim();

            if (title === '') {
                updateFeedback('Judul tugas tidak boleh kosong.', 'error');
                return;
            }

            // --- Cek Duplikasi Sebelum Tambah ---
            if (isDuplicate(title)) {
                updateFeedback(`Tugas "${title}" sudah ada!`, 'error');
                return; // Hentikan proses penambahan
            }

            // Jika lolos, tambahkan tugas baru
            const newId = tasks.length > 0 ? tasks[tasks.length - 1].id + 1 : 1;
            const newTask = { id: newId, title: title, isCompleted: false };

            // Perbarui state dan render ulang
            tasks = [...tasks, newTask];
            renderTasks();
            updateFeedback(`Tugas "${title}" berhasil ditambahkan.`);
            newTaskTitleInput.value = '';
        }

        // --- INISIALISASI ---
        document.addEventListener('DOMContentLoaded', () => {
            renderTasks();
            document.getElementById('task-form').addEventListener('submit', handleAddTask);
        });
    </script>
</body>
</html>
