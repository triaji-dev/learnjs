16. Date Object & Formatting (15 Menit)

A. Konsep Dasar Date Object

Objek Date di JavaScript digunakan untuk bekerja dengan tanggal dan waktu. Ini adalah objek yang kompleks karena harus menangani zona waktu, perbedaan bulan, dan tahun kabisat.

Membuat Date Object

Ada beberapa cara untuk menginisialisasi objek Date:

Cara InisialisasiContohDeskripsi
Current Timenew Date()Membuat objek Date untuk waktu dan tanggal saat ini.
Timestamp (ms)new Date(0)Membuat objek Date dari jumlah milidetik sejak Epoch Time (1 Januari 1970 UTC).
Date Stringnew Date('2023-12-01')Membuat objek dari string tanggal (seperti yang kita dapatkan dari input form atau API).

Mengambil Data Dasar

Setelah objek dibuat, kita dapat mengekstrak bagian-bagiannya:

JavaScript

const today = new Date();

console.log(today.getFullYear()); // 2025
console.log(today.getMonth());    // 0-11 (Januari = 0, Desember = 11)
console.log(today.getDate());     // Tanggal dalam bulan (1-31)
console.log(today.getDay());      // Hari dalam seminggu (0=Minggu, 6=Sabtu)

Penting: Perlu diperhatikan bahwa getMonth() berbasis nol (0-11), yang seringkali menjadi sumber bug bagi pengembang.

B. Format Deadline (Locale & Options)

Mengubah timestamp mentah ("2023-12-01") menjadi format yang cantik ("Jumat, 1 Des") secara manual adalah pekerjaan yang rumit (membutuhkan array nama bulan dan hari, plus penanganan bahasa).

Solusi modern dan terbaik di JavaScript adalah menggunakan Intl.DateTimeFormat melalui metode toLocaleDateString(). Metode ini secara otomatis menangani zona waktu dan bahasa lokal.

Metode toLocaleDateString()

Metode ini menerima dua argumen:

  1. locale: Kode bahasa yang diinginkan (misalnya, 'id-ID', 'en-US').
  1. options: Objek yang menentukan bagaimana elemen tanggal harus ditampilkan.

Contoh Penerapan Format Cantik:

Kita ingin format seperti "Jumat, 1 Desember 2023"

JavaScript

const rawDateString = '2023-12-01'; // Data dari API/Form
const dateObject = new Date(rawDateString);

const options = {
    weekday: 'long', // long, short, narrow
    year: 'numeric', // numeric, 2-digit
    month: 'short',  // numeric, 2-digit, long, short, narrow
    day: 'numeric'   // numeric, 2-digit
};

const formattedDate = dateObject.toLocaleDateString('id-ID', options);
// Hasil: "Jumat, 1 Des 2023"

console.log(formattedDate);

Keuntungan Menggunakan toLocaleDateString():

  • Internasionalisasi: Otomatis menyesuaikan urutan tanggal dan nama bulan/hari sesuai locale.
  • Keterbacaan: Kode jauh lebih bersih dan tidak memerlukan logika if/else yang rumit untuk memetakan angka ke nama bulan.

C. Relevansi React: Mengolah Data Display di UI

Dalam React, data yang diambil dari state (atau props) harus diproses menjadi format yang ramah pengguna sebelum ditampilkan.

Prinsip:

  1. Data mentah (raw data) harus disimpan di state (misalnya, string 2023-12-01).
  1. Pemrosesan dan pemformatan (menggunakan new Date() dan toLocaleDateString()) dilakukan hanya saat data akan ditampilkan di render.

Ini memastikan bahwa data mentah tetap konsisten untuk operasi logis (seperti membandingkan tanggal) sementara pengguna melihat format yang indah.

JavaScript

function TaskItem({ task }) {
    // task.dueDate adalah string mentah ("2023-12-01")

    // Pemrosesan terjadi saat render
    const dateObj = new Date(task.dueDate);
    const options = { month: 'short', day: 'numeric' };
    const displayDate = dateObj.toLocaleDateString('en-US', options);

    return (
        <div className="task">
            <span>{task.title}</span>
            <span className="deadline">Deadline: {displayDate}</span>
        </div>
    );
}

Live Coding: Format Deadline Tugas (15 Menit)

Kita akan membuat fitur TaskMaster yang mengambil deadline mentah dan menampilkannya dalam berbagai format "cantik" menggunakan toLocaleDateString().

Exercise untuk peserta:

  1. Buat array tugas dengan properti dueDate yang berisi string format ISO (YYYY-MM-DD).
  1. Buat fungsi formatDate(dateString, options) yang membungkus logika new Date() dan toLocaleDateString().
  1. Di dalam fungsi rendering, panggil formatDate untuk setiap tugas dengan options yang berbeda (misalnya, satu untuk format panjang, satu untuk format pendek).

Contoh Code Starter (HTML/JS):

HTML

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Praktek: Date Object & Formatting</title>
    <script src="https://cdn.tailwindcss.com"></script>
</head>
<body class="bg-gray-100 p-8">

    <div id="app" class="max-w-xl mx-auto bg-white p-6 rounded-lg shadow-xl">
        <h1 class="text-3xl font-bold mb-6 text-gray-800">TaskMaster: Deadline Formatting</h1>
        
        <ul id="task-list" class="space-y-4">
            </ul>
    </div>

    <script>
        // --- DATA DENGAN TANGGAL MENTAH ---
        const TASKS_WITH_DEADLINE = [
            { id: 1, title: 'Finalisasi Proyek', dueDate: '2025-11-28', isCompleted: false },
            { id: 2, title: 'Kirim Invoice Klien', dueDate: '2025-12-01', isCompleted: false },
            { id: 3, title: 'Review Code Base', dueDate: '2025-11-23', isCompleted: true },
        ];

        const taskListContainer = document.getElementById('task-list');
        
        // --- FUNGSI UTAMA: PEMFORMATAN TANGGAL ---
        /**
         * Mengubah string tanggal mentah menjadi format cantik menggunakan Intl.
         * @param {string} dateString - Tanggal dalam format YYYY-MM-DD.
         * @param {object} options - Opsi pemformatan untuk toLocaleDateString.
         */
        function formatDate(dateString, options) {
            // 1. Buat Date Object dari string mentah
            const dateObject = new Date(dateString);
            
            // 2. Gunakan toLocaleDateString() dengan locale Indonesia (id-ID)
            return dateObject.toLocaleDateString('id-ID', options);
        }

        // --- FUNGSI RENDER ---
        function renderTasks() {
            taskListContainer.innerHTML = '';

            TASKS_WITH_DEADLINE.forEach(task => {
                
                // --- CONTOH 1: FORMAT PANJANG (Untuk Detail View) ---
                const longOptions = { weekday: 'long', year: 'numeric', month: 'long', day: 'numeric' };
                const longFormat = formatDate(task.dueDate, longOptions);

                // --- CONTOH 2: FORMAT PENDEK (Untuk List Item) ---
                const shortOptions = { day: 'numeric', month: 'short' };
                const shortFormat = formatDate(task.dueDate, shortOptions);

                const statusColor = task.isCompleted ? 'bg-green-100 text-green-800' : 'bg-red-100 text-red-800';
                
                const taskItemHTML = `
                    <li class="p-4 border rounded-lg flex justify-between items-center bg-white">
                        <div>
                            <span class="font-medium text-lg ${task.isCompleted ? 'line-through text-gray-500' : 'text-gray-900'}">${task.title}</span>
                            <p class="text-sm text-gray-600 mt-0.5">
                                Deadline: <span class="font-semibold text-indigo-700">${shortFormat}</span> (${longFormat})
                            </p>
                        </div>
                        <span class="px-3 py-1 text-xs font-semibold rounded-full ${statusColor}">
                            ${task.isCompleted ? 'Selesai' : 'Pending'}
                        </span>
                    </li>
                `;
                taskListContainer.innerHTML += taskItemHTML;
            });
        }

        // --- INISIALISASI ---
        document.addEventListener('DOMContentLoaded', renderTasks);
    </script>
</body>
</html>
