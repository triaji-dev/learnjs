A. Konsep Dasar Arrow Functions

Arrow Functions adalah sintaks ringkas (concise syntax) untuk menulis fungsi di JavaScript yang diperkenalkan pada ES6 (ECMAScript 2015). Mereka seringkali lebih disukai daripada fungsi tradisional (function keyword) karena kodenya lebih pendek dan memiliki perilaku yang lebih konsisten terkait dengan konteks this.

1. Sintaks Dasar dan Perbandingan

FiturFungsi TradisionalArrow Function
Sintaks Penuhfunction add(a, b) { return a + b; }const add = (a, b) => { return a + b; };
Sintaks Ringkas (Implisit Return)(Tidak ada)const add = (a, b) => a + b;
Satu Parameterfunction greet(name) { ... }const greet = name => { ... }; (Tanda kurung () bisa dihilangkan)
Tanpa Parameterfunction sayHi() { ... }const sayHi = () => { ... };

2. Implisit Return (Return Tersirat)

Fitur kunci yang akan kita gunakan untuk membuat komponen kartu tugas adalah Implisit Return.

  • Jika body fungsi hanya terdiri dari satu ekspresi tunggal (tidak perlu kurung kurawal {}), Anda dapat menghilangkan kata kunci return. Nilai dari ekspresi tunggal tersebut secara otomatis akan menjadi nilai yang dikembalikan oleh fungsi.
FungsiSintaks
Explicit Return (Dinyatakan)const calculateArea = (w, h) => { return w * h; };
Implicit Return (Tersirat)const calculateArea = (w, h) => w * h;

3. Mengembalikan Objek atau Struktur HTML

Ketika Anda ingin mengembalikan sebuah Objek Literal atau String HTML Multi-baris (seperti yang dibutuhkan untuk Kartu Tugas), Anda harus menggunakan kurung kurawal ({}) yang diapit oleh tanda kurung (()) untuk membedakannya dari body fungsi biasa.

  • Mengembalikan Objek:JavaScript
    // SALAH: JS akan mengira {} adalah body fungsi, bukan objek literal
const getPerson = (name, age) => { name: name, age: age }; 

// BENAR: Menggunakan () untuk mengapit objek literal
const getPerson = (name, age) => ({ name: name, age: age });
  • Mengembalikan String HTML Multi-baris (Implisit Return):JavaScript

    Ini sangat berguna saat kita mengombinasikan Arrow Functions dengan Template Literals (```) dari sesi sebelumnya.

    const task = { title: "Deploy App", priority: "High" };
// Implisit Return dari Template Literal
const renderCard = (task) => (
    `
    <div class="task-card ${task.priority.toLowerCase()}">
        <h3>${task.title}</h3>
        <p>Priority: ${task.priority}</p>
    </div>
    `
);

console.log(renderCard(task)); 
// Output adalah string HTML yang sudah terisi data

Live Coding (15 Menit)

Kita akan membuat fungsi createTaskCard(task) yang menerima objek tugas dan mengembalikan string HTML lengkap kartu tugas, memanfaatkan:

  1. Arrow Function dengan Implisit Return.
  1. Template Literals untuk menyusun markup HTML.

Tujuan Live Coding:

  • Mengubah fungsi tradisional menjadi Arrow Function.
  • Mengimplementasikan Implisit Return untuk mengembalikan string HTML.
  • Menerima dan menyisipkan data dari objek task ke dalam string HTML.
  • Membuat variasi tampilan (misalnya, warna) berdasarkan properti data (misalnya, priority).

Contoh Kode:

Kita akan melanjutkan dari file HTML sebelumnya (dengan Tailwind CSS CDN).

HTML

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Praktek: Arrow Functions & Return</title>
    <script src="https://cdn.tailwindcss.com"></script>
    <style>
        .priority-high { border-left: 4px solid #ef4444; } /* Merah */
        .priority-medium { border-left: 4px solid #f97316; } /* Jingga */
        .priority-low { border-left: 4px solid #10b981; } /* Hijau */
    </style>
</head>
<body>

    <div id="app" class="p-8 max-w-2xl mx-auto">
        <h1 class="text-3xl font-bold mb-6 text-gray-800">Daftar Tugas TaskMaster</h1>
        <div id="task-list" class="space-y-4">
            </div>
    </div>

    <script>
        // --- DATA STATIS ---
        const TASK_LIST = [
            { id: 1, title: "Revisi Desain Landing Page", priority: "High", isDone: false },
            { id: 2, title: "Review Code Feature X", priority: "Medium", isDone: false },
            { id: 3, title: "Update Dokumentasi API", priority: "Low", isDone: true },
        ];
        
        // --- ARROW FUNCTION DENGAN IMPLISIT RETURN ---
        
        // Fungsi untuk mengembalikan string class CSS berdasarkan prioritas
        const getPriorityClass = priority => `priority-${priority.toLowerCase()}`;
        
        /**
         * @param {Object} task - Objek tugas { title, priority, isDone }
         * @returns {string} String HTML untuk kartu tugas
         */
        // Menggunakan Arrow Function dan IMPLISIT RETURN untuk mengembalikan Template Literal HTML
        const createTaskCard = (task) => (
            `
            <div class="bg-white p-4 shadow-md rounded-lg ${getPriorityClass(task.priority)} flex justify-between items-center">
                <div class="${task.isDone ? 'line-through text-gray-500' : 'text-gray-800'}">
                    <h3 class="text-lg font-semibold">${task.title}</h3>
                    <p class="text-sm text-gray-600">Prioritas: ${task.priority}</p>
                </div>
                <div class="px-3 py-1 text-xs font-bold rounded-full 
                    ${task.isDone ? 'bg-green-200 text-green-800' : 'bg-red-200 text-red-800'}">
                    ${task.isDone ? 'SELESAI' : 'BELUM'}
                </div>
            </div>
            `
        );


        // --- FUNGSI UTAMA UNTUK MERENDER SEMUA TUGAS ---
        function renderTaskList() {
            const taskListContainer = document.getElementById('task-list');
            
            // Menggunakan .map() pada array untuk mengkonversi setiap objek menjadi string HTML
            // dan .join('') untuk menggabungkan semua string HTML menjadi satu blok
            const allTaskHTML = TASK_LIST.map(task => createTaskCard(task)).join('');
            
            taskListContainer.innerHTML = allTaskHTML;
        }

        // Jalankan fungsi setelah DOM dimuat
        document.addEventListener('DOMContentLoaded', renderTaskList);

    </script>
</body>
</html>

Langkah-langkah di Live Coding:

  1. Demonstrasi createTaskCard: Tunjukkan bagaimana fungsi menerima task dan dengan cepat me-return string HTML tanpa kata kunci return.
  1. Modifikasi Implisit Return: Coba hilangkan tanda kurung () yang mengapit Template Literal dan tunjukkan error yang terjadi (JavaScript mengira itu adalah body fungsi biasa).
  1. Penggunaan .map(): Jelaskan bagaimana fungsi createTaskCard digunakan di dalam metode TASK_LIST.map() untuk mengubah array data menjadi array HTML. Ini adalah konsep fundamental dalam rendering list di React.
  1. Menambahkan Logika: Ajak peserta untuk mengubah warna kartu atau menambahkan ikon berdasarkan status isDone atau priority lainnya, mempraktikkan ekspresi di dalam Template Literal dan Arrow Function.
