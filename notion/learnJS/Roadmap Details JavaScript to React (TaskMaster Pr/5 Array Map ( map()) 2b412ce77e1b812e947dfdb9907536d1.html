1. Penjelasan Materi dan Contoh (15 Menit)

A. Konsep Dasar Array Map

Metode .map() adalah salah satu metode Higher-Order Function pada Array di JavaScript. Tujuan utamanya adalah untuk melakukan iterasi (perulangan) pada setiap elemen dalam sebuah Array dan mengembalikan Array baru yang berisi hasil dari pemrosesan setiap elemen tersebut.

Tiga Poin Kunci:

  1. Iterasi: .map() mengunjungi setiap elemen dalam Array sumber.
  1. Transformasi: Ia menerapkan fungsi callback yang Anda sediakan pada setiap elemen.
  1. Array Baru: Ia selalu mengembalikan Array baru dengan jumlah elemen yang sama. Array sumber tidak diubah (non-mutating).

Sintaks:

JavaScript

const newArray = originalArray.map((element, index, array) => {
    // Kembalikan nilai yang ditransformasi
    return transformedValue;
});

B. Contoh 1: Transformasi Data Sederhana

Misalnya, kita punya daftar angka dan ingin mengalikannya dengan dua.

JavaScript

const numbers = [1, 2, 3, 4];

// Transformasi: Setiap angka dikalikan 2
const doubledNumbers = numbers.map(number => {
    return number * 2;
});

console.log(doubledNumbers); // Output: [2, 4, 6, 8]
console.log(numbers);        // Array sumber tidak berubah: [1, 2, 3, 4]

C. Detail Implementasi: Array of Objects ke Markup HTML

Ini adalah skenario penggunaan terpenting di pengembangan Front-End modern (seperti React) â€” mengubah Array of Objects yang mewakili data (misalnya, daftar tugas) menjadi daftar elemen HTML yang dapat ditampilkan di antarmuka pengguna.

Contoh 2: Rendering Daftar Tugas (Fitur TaskMaster)

Kita memiliki daftar tugas dalam bentuk Array objek dan ingin mengubahnya menjadi string HTML untuk setiap kartu tugas.

JavaScript

const taskList = [
    { id: 1, title: 'Beli Bahan Makanan', priority: 'Low' },
    { id: 2, title: 'Perbaiki Bug Login', priority: 'High' },
    { id: 3, title: 'Rapat Mingguan Tim', priority: 'Medium' }
];

// Menggunakan .map() untuk mengubah setiap objek tugas menjadi string HTML
const taskCardsHTML = taskList.map(task => {
    // Di sini kita menggunakan Template Literals untuk membuat markup
    const priorityClass = task.priority === 'High' ? 'text-red-600' : 'text-gray-600';
    
    return `
        <div class="task-card border p-3 mb-2 rounded shadow-sm">
            <h3 class="text-lg font-semibold">${task.title}</h3>
            <p class="${priorityClass}">Prioritas: ${task.priority}</p>
        </div>
    `;
});

console.log(taskCardsHTML); 
/* Output: 
Array dari 3 string HTML: 
[
    "<div class=... >...</div>", 
    "<div class=... >...</div>", 
    "<div class=... >...</div>"
]
*/

// Catatan: Array of strings ini perlu digabungkan (.join('')) sebelum disisipkan ke DOM.
const finalHTML = taskCardsHTML.join(''); 
// Output: Satu string panjang yang siap disisipkan ke innerHTML

Exercise untuk Peserta

Ajak peserta untuk mengimplementasikan fungsionalitas Render Daftar Tugas menggunakan .map(), Template Literals, dan konsep Destructuring yang sudah dipelajari.

  1. Siapkan Array Data Tugas: Buat Array bernama taskData yang berisi minimal 3 objek tugas (dengan id, title, priority, dan status).
  1. Tulis Fungsi renderTask: Buat fungsi yang menerima satu objek tugas dan mengembalikan string HTML kartu tugas yang sudah diformat (gunakan Destructuring dan Template Literals).
  1. Terapkan .map(): Gunakan .map() pada taskData untuk memanggil fungsi renderTask pada setiap elemen.
  1. Gabungkan dan Tampilkan: Gunakan metode .join('') untuk mengubah Array hasil menjadi satu string besar, kemudian sisipkan ke DOM menggunakan innerHTML.

        // -----------------------------------------------------------------

                            <button class="text-sm text-indigo-600 hover:text-indigo-800 font-medium">Lanjutkan</button>
                        </div>`
                    }
                </div>

        const taskCardsArrayOfStrings = taskData.map(task => {
            // Memanggil fungsi pembantu untuk setiap objek tugas
            return renderTaskCard(task);
