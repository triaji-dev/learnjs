7. Events & Callbacks (15 Menit)

A. Konsep Dasar Event Handling

Dalam pengembangan web modern, Event adalah sinyal dari browser bahwa sesuatu telah terjadi. Ini bisa berupa interaksi pengguna (seperti mengklik tombol) atau pembaruan browser (seperti halaman yang selesai dimuat).

Contoh Event yang umum:

  • click: Ketika tombol mouse diklik di atas elemen.
  • submit: Ketika formulir dikirimkan.
  • mouseover: Ketika kursor mouse memasuki elemen.
  • keydown: Ketika tombol keyboard ditekan.
  • load: Ketika seluruh halaman/sumber daya telah dimuat.

Event Listener

Untuk merespons sebuah event, kita menggunakan Event Listener. Listener adalah sebuah fungsi (disebut Event Handler) yang "mendengarkan" dan akan dieksekusi hanya ketika event yang spesifik terjadi pada elemen tertentu.

Mekanisme:

  1. Target: Tentukan elemen HTML yang ingin Anda pantau (misalnya, sebuah tombol "Delete").
  1. Event Type: Tentukan jenis event yang ditunggu (misalnya, click).
  1. Handler Function: Pasang fungsi yang akan dijalankan saat event terjadi.

Contoh JavaScript Murni (addEventListener):

JavaScript

// 1. Dapatkan elemen target
const deleteButton = document.getElementById('delete-task-btn');

// 2. Tambahkan event listener untuk event 'click'
deleteButton.addEventListener('click', function() {
    // 3. Handler Function (Kode yang akan dieksekusi)
    console.log('Tombol Delete telah diklik!');
    // Logika untuk menghapus tugas ada di sini
});

B. Memahami Callback Function

Callback Function adalah fungsi yang dilewatkan sebagai argumen ke fungsi lain dan dieksekusi nanti. Konsep ini adalah fondasi dari asynchronous programming dan event handling di JavaScript.

Relevansi dengan Event Handling

Dalam konteks event handling, fungsi yang Anda berikan ke addEventListener (atau properti onclick di React) adalah sebuah callback. Fungsi ini tidak dieksekusi segera; ia ditelepon kembali (called back) oleh browser hanya setelah event terkait terjadi.

Struktur Event Handler sebagai Callback:

JavaScript

// Function A (Fungsi Tingkat Tinggi)
function addEventListener(eventType, callbackFunction) {
    // browser akan menunggu eventType terjadi...
    // Ketika event terjadi:
    callbackFunction(); // <-- Callback dipanggil
}

// Function B (Callback yang akan dipanggil kembali)
function hapusTugas() {
    alert("Tugas berhasil dihapus!");
}

// Implementasi
deleteButton.addEventListener('click', hapusTugas); // `hapusTugas` adalah callback

Studi Kasus: Mengoper Fungsi sebagai Callback

Seperti yang disebutkan dalam screenshot ("Fungsi delete dipassing sebagai parameter callback"), ini sangat relevan dalam arsitektur komponen modern (seperti React) di mana Anda sering kali perlu mengoper fungsi dari komponen induk ke komponen anak melalui props.

Skenario "TaskMaster":

  • Komponen Induk (<TaskList>): Tempat data tugas disimpan dan logika deleteTask berada.
  • Komponen Anak (<DeleteButton>): Tombol yang hanya bertanggung jawab untuk memberikan sinyal bahwa ia diklik.

Tujuan: Saat tombol di Komponen Anak diklik, Komponen Induk harus menjalankan fungsi deleteTask miliknya.

Penerapan Konsep Callback:

  1. Komponen Induk membuat fungsi deleteTask.
  1. Komponen Induk mengoper fungsi deleteTask ini ke Komponen Anak sebagai sebuah prop (misalnya, bernama onDelete).
  1. Komponen Anak menggunakan prop onDelete tersebut sebagai callback untuk event click pada tombol.

C. Relevansi React: Event Handling (onClick)

Di React, event handling mengikuti prinsip yang sama tetapi dengan sintaks yang sedikit berbeda, yang dikenal sebagai Synthetic Events.

  • Penulisan: Nama event menggunakan camelCase (misalnya, onClick, onChange) dan bukan lowercase seperti di HTML (onclick, onchange).
  • Callback Langsung: Anda memberikan fungsi JavaScript (callback) langsung ke prop onClick.

JavaScript

// Komponen Anak (The Button)
function DeleteButton({ taskId, onDelete }) {
    // Saat tombol diklik, panggil callback onDelete
    // dan oper ID tugas yang akan dihapus.
    return (
        <button onClick={() => onDelete(taskId)}>
            Hapus Tugas
        </button>
    );
}

// Komponen Induk (The Parent)
function TaskManager() {
    // 1. Fungsi delete adalah Logika di Induk
    const deleteTask = (id) => {
        console.log(`Menghapus tugas dengan ID: ${id}`);
        // Lakukan pembaruan state di sini
    };
    
    // 2. Mengoper fungsi delete ke Anak sebagai prop 'onDelete'
    return (
        <DeleteButton taskId="task-101" onDelete={deleteTask} />
    );
}

Dalam contoh di atas, deleteTask adalah callback yang dioper sebagai prop dari TaskManager ke DeleteButton. Saat onClick diaktifkan di DeleteButton, ia memanggil callback tersebut.

Live Coding: Menerapkan Events & Callbacks (15 Menit)

Dalam sesi ini, kita akan membuat sebuah aplikasi TaskMaster sederhana di JavaScript murni (bukan React) untuk menunjukkan bagaimana event listener dan callback bekerja untuk menghapus sebuah item.

Tujuan Live Coding:

  • Membuat daftar tugas sederhana di HTML.
  • Menambahkan tombol Delete pada setiap tugas.
  • Menerapkan addEventListener pada tombol tersebut.
  • Menggunakan callback function untuk menjalankan logika penghapusan yang dinamis (mengetahui tugas mana yang harus dihapus).

Exercise untuk peserta:

  1. Buat struktur HTML dengan beberapa item tugas dan tombol Delete di setiap item.
  1. Tulis fungsi deleteTask(taskId) yang menerima ID tugas dan hanya me-log pesan.
  1. Tulis fungsi setupEventListeners yang dipanggil saat halaman dimuat.
  1. Di dalam setupEventListeners, gunakan document.querySelectorAll untuk memilih semua tombol Delete.
  1. Iterasi (forEach) pada setiap tombol dan pasang click listener.
  1. Gunakan fungsi anonim/arrow function sebagai callback pada addEventListener untuk memanggil deleteTask dengan ID tugas yang benar (Anda bisa mendapatkan ID dari atribut data HTML, misalnya data-task-id).
  1. (Tantangan Opsional): Ubah fungsi deleteTask agar benar-benar menghapus elemen tugas dari DOM.

Contoh Code Starter (HTML/JS):

HTML

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Praktek: Events & Callbacks</title>
    <script src="https://cdn.tailwindcss.com"></script>
</head>
<body class="bg-gray-100 p-8">

    <div id="app" class="max-w-xl mx-auto bg-white p-6 rounded-lg shadow-lg">
        <h1 class="text-3xl font-bold mb-6 text-gray-800">TaskMaster Sederhana</h1>
        
        <ul id="task-list" class="space-y-4">
            <li class="flex justify-between items-center p-3 bg-indigo-50 rounded-lg" data-task-id="task-1">
                <span class="text-lg font-medium">Buat Laporan Bulanan</span>
                <button class="delete-btn bg-red-500 text-white px-3 py-1 rounded-md hover:bg-red-600 transition duration-150" data-task-id="task-1">
                    Delete
                </button>
            </li>
            
            <li class="flex justify-between items-center p-3 bg-indigo-50 rounded-lg" data-task-id="task-2">
                <span class="text-lg font-medium">Review Code Project X</span>
                <button class="delete-btn bg-red-500 text-white px-3 py-1 rounded-md hover:bg-red-600 transition duration-150" data-task-id="task-2">
                    Delete
                </button>
            </li>
        </ul>
        
        <p id="status-message" class="mt-6 text-center text-gray-500"></p>
    </div>

    <script>
        // --- LANGKAH 1: Fungsi Callback (Logika Penghapusan) ---
        function deleteTask(taskId) {
            console.log(`[CALLBACK DIPANGGIL] Mencoba menghapus tugas dengan ID: ${taskId}`);
            const taskItem = document.querySelector(`[data-task-id="${taskId}"]`);
            
            if (taskItem) {
                // Hapus elemen dari DOM (Tantangan Opsional)
                taskItem.remove(); 
                
                // Update pesan status
                document.getElementById('status-message').textContent = `Tugas "${taskItem.querySelector('span').textContent}" berhasil dihapus.`;
            } else {
                console.error(`Elemen tugas dengan ID ${taskId} tidak ditemukan.`);
            }
        }

        // --- LANGKAH 2: Setup Event Listeners ---
        function setupEventListeners() {
            console.log("Memasang Event Listeners...");
            
            // Pilih semua tombol dengan class 'delete-btn'
            const deleteButtons = document.querySelectorAll('.delete-btn');

            // Iterasi dan pasang listener ke setiap tombol
            deleteButtons.forEach(button => {
                // Dapatkan ID tugas dari atribut data
                const taskId = button.getAttribute('data-task-id');
                
                // Tambahkan event listener. Fungsi di bawah adalah CALLBACK
                button.addEventListener('click', function() {
                    // Panggil fungsi callback utama dengan ID yang sudah didapatkan
                    deleteTask(taskId); 
                });
            });
        }

        // Jalankan fungsi setup setelah DOM selesai dimuat
        document.addEventListener('DOMContentLoaded', setupEventListeners);
    </script>
</body>
</html>
