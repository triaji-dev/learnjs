8. Array Filter (.filter()) (15 Menit)

A. Konsep Dasar Array Method: .filter()

Metode .filter() adalah Higher-Order Function (fungsi tingkat tinggi) bawaan JavaScript yang digunakan untuk membuat array baru yang berisi semua elemen dari array asli yang lolos uji (mengembalikan true) dari fungsi callback yang disediakan.

Prinsip Utama Immutability (Ketidakmampuan Berubah)

Sama seperti .map(), .filter() adalah metode non-mutating. Artinya, ia tidak mengubah array asli. Ini sangat penting dalam pemrograman modern (terutama di React) karena menjamin bahwa operasi pemrosesan data selalu menghasilkan data baru, sehingga mudah dilacak dan dihindari efek samping yang tidak diinginkan.

Sintaks Dasar

.filter() menerima sebuah fungsi callback. Fungsi callback ini harus mengembalikan nilai Boolean (true atau false) untuk setiap elemen array.

JavaScript

const newArray = array.filter((element, index, array) => {
    // Kembalikan TRUE jika elemen HARUS dimasukkan ke newArray
    // Kembalikan FALSE jika elemen HARUS DIBUANG
    return element.property > value; 
});
  • element: (Wajib) Nilai elemen array saat ini yang sedang diproses.
  • index: (Opsional) Indeks dari elemen saat ini.
  • array: (Opsional) Array yang dipanggil.

B. Implementasi di Fitur "TaskMaster"

.filter() adalah alat utama untuk dua fitur inti dalam aplikasi TaskMaster:

1. Menghapus Tugas (Menghasilkan Array Baru Tanpa ID Tersebut)

Menghapus item dari array tugas tidak dilakukan dengan menghapus item secara literal dari array asli. Sebaliknya, kita membuat array baru yang melompati item yang ID-nya ingin dihapus.

Contoh: Kita ingin menghapus tugas dengan id: 't-2'.

JavaScript

const initialTasks = [
    { id: 't-1', title: 'Belajar .filter()', isCompleted: false },
    { id: 't-2', title: 'Hapus Tugas Lama', isCompleted: true }, // <--- INI AKAN DIBUANG
    { id: 't-3', title: 'Live Coding', isCompleted: false },
];

const taskIdToDelete = 't-2';

// Filter akan mengembalikan TRUE untuk SEMUA tugas, kecuali yang ID-nya 't-2'
const remainingTasks = initialTasks.filter(task => {
    return task.id !== taskIdToDelete; // Hanya yang TIDAK SAMA dengan 't-2' yang akan lolos
});

/*
Hasil (remainingTasks):
[
    { id: 't-1', title: 'Belajar .filter()', isCompleted: false },
    { id: 't-3', title: 'Live Coding', isCompleted: false },
] 
*/

2. Filter Tab "Selesai" vs "Belum Selesai"

Ini adalah penggunaan filter() yang paling umum: mengambil subset (bagian kecil) dari data berdasarkan kondisi properti.

A. Filter Tugas yang Belum Selesai:

JavaScript

// Filter yang mengembalikan hanya tugas yang isCompleted: false
const activeTasks = initialTasks.filter(task => {
    return task.isCompleted === false; 
});

// Atau disingkat:
// const activeTasks = initialTasks.filter(task => !task.isCompleted);

B. Filter Tugas yang Sudah Selesai:

JavaScript

// Filter yang mengembalikan hanya tugas yang isCompleted: true
const completedTasks = initialTasks.filter(task => {
    return task.isCompleted === true;
});

// Atau disingkat:
// const completedTasks = initialTasks.filter(task => task.isCompleted);

C. Relevansi React: State Update & Derived State

State Update (Immutability)

Di React (menggunakan useState), state harus diperbarui secara imutable (tidak mengubah objek/array secara langsung).

Ketika Anda memiliki array tugas di dalam state: const [tasks, setTasks] = useState([...]), Anda HARUS menggunakan .filter() untuk membuat array baru saat menghapus tugas.

JavaScript

// Di dalam komponen React:
const handleDelete = (taskId) => {
    // ❌ SALAH: Mutasi state langsung
    // tasks.splice(index, 1); 
    // setTasks(tasks); // React tidak akan me-render ulang

    // ✅ BENAR: Menggunakan .filter() untuk membuat array baru
    const newTasks = tasks.filter(task => task.id !== taskId);
    setTasks(newTasks); // React melihat array baru dan me-render ulang
};

Derived State (State Turunan)

Derived State adalah data yang dihitung dari state yang sudah ada. Fitur "filter tab" adalah contoh sempurna dari derived state.

Daripada menyimpan tiga state (semua tugas, tugas aktif, tugas selesai), kita hanya perlu menyimpan satu state utama (tasks) dan satu state filter (filterType).

JavaScript

function TaskListApp() {
    const [tasks, setTasks] = useState(initialTasks);
    const [filterType, setFilterType] = useState('all'); // 'all', 'active', 'completed'

    // Derived State: Array yang ditampilkan dihitung setiap kali state berubah
    const filteredTasks = tasks.filter(task => {
        if (filterType === 'active') {
            return !task.isCompleted;
        }
        if (filterType === 'completed') {
            return task.isCompleted;
        }
        return true; // filterType === 'all'
    });
    
    // ... Render filteredTasks ...
}

Live Coding: Fitur Hapus & Filter Tugas (15 Menit)

Dalam sesi ini, kita akan membuat sebuah TaskMaster sederhana di JavaScript murni untuk menerapkan logika .filter() untuk menghapus item dan memfilter daftar tugas.

Exercise untuk peserta:

  1. Buat array objek tugas statis.
  1. Tulis fungsi renderTasks() untuk menampilkan array tugas ke HTML.
  1. Tulis fungsi deleteTask(taskId) yang menggunakan .filter() untuk membuat array baru tanpa tugas tersebut, lalu panggil renderTasks() lagi.
  1. Tulis fungsi filterTasks(filterType) yang menggunakan .filter() untuk memilih hanya tugas yang isCompleted: true atau isCompleted: false, lalu panggil renderTasks() lagi.
  1. Hubungkan fungsi deleteTask ke tombol Delete, dan filterTasks ke tombol filter ("Aktif" / "Selesai").

Contoh Code Starter (HTML/JS):

HTML

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Praktek: Array Filter (.filter())</title>
    <script src="https://cdn.tailwindcss.com"></script>
</head>
<body class="bg-gray-100 p-8">

    <div id="app" class="max-w-xl mx-auto bg-white p-6 rounded-lg shadow-lg">
        <h1 class="text-3xl font-bold mb-6 text-gray-800">TaskMaster Filter Demo</h1>
        
        <div class="flex space-x-2 mb-6 border-b pb-4">
            <button data-filter="all" class="filter-btn bg-indigo-500 text-white px-4 py-2 rounded-md hover:bg-indigo-600">Semua</button>
            <button data-filter="active" class="filter-btn bg-gray-200 text-gray-800 px-4 py-2 rounded-md hover:bg-gray-300">Aktif</button>
            <button data-filter="completed" class="filter-btn bg-gray-200 text-gray-800 px-4 py-2 rounded-md hover:bg-gray-300">Selesai</button>
        </div>

        <ul id="task-list" class="space-y-3">
            </ul>
        
        <p id="status-message" class="mt-6 text-center text-gray-500"></p>
    </div>

    <script>
        // --- DATA STATE APLIKASI ---
        let tasks = [
            { id: 1, title: 'Belajar JavaScript Modern', isCompleted: false },
            { id: 2, title: 'Siapkan Presentasi Materi', isCompleted: true },
            { id: 3, title: 'Review PR Teman', isCompleted: false },
            { id: 4, title: 'Lunch Break', isCompleted: true },
        ];
        let currentFilter = 'all'; // State untuk filter aktif

        const taskListContainer = document.getElementById('task-list');
        const statusMessage = document.getElementById('status-message');

        // --- FUNGSI UTAMA: RENDER TUGAS ---
        function renderTasks(tasksToRender) {
            // 1. Clear container
            taskListContainer.innerHTML = ''; 

            if (tasksToRender.length === 0) {
                taskListContainer.innerHTML = `<li class="text-center p-4 text-gray-500 italic">Tidak ada tugas dalam filter ini.</li>`;
                return;
            }

            // 2. Loop dan sisipkan HTML
            tasksToRender.forEach(task => {
                const isChecked = task.isCompleted ? 'line-through text-gray-500' : 'text-gray-900';
                
                // Gunakan Template Literals untuk membuat UI tugas
                const taskItemHTML = `
                    <li class="flex justify-between items-center p-3 bg-white border rounded-lg shadow-sm">
                        <span class="text-lg font-medium ${isChecked}">${task.title}</span>
                        <button data-id="${task.id}" class="delete-btn bg-red-500 text-white px-3 py-1 text-sm rounded-md hover:bg-red-600 transition duration-150">
                            Hapus
                        </button>
                    </li>
                `;
                taskListContainer.innerHTML += taskItemHTML;
            });
            
            // Re-setup delete listeners karena elemen baru
            setupDeleteListeners();
        }

        // --- LOGIKA 1: FUNGSI DELETE (Menggunakan .filter() untuk membuat array baru) ---
        function deleteTask(taskId) {
            console.log(`Menghapus tugas ID: ${taskId}`);

            // ✅ Gunakan .filter() untuk membuat array baru TANPA tugas dengan ID ini
            // Ini menjamin Immutability (data asli tidak diubah)
            tasks = tasks.filter(task => task.id !== taskId);

            // Setelah data state diubah, render ulang daftar tugas
            filterTasks(currentFilter); // Render array tasks yang baru difilter
            statusMessage.textContent = `Tugas ID ${taskId} berhasil dihapus.`;
        }
        
        // --- LOGIKA 2: FUNGSI FILTER (Menggunakan .filter() untuk Derived State) ---
        function filterTasks(filterType) {
            currentFilter = filterType; // Update state filter
            let filteredArray = tasks;
            
            if (filterType === 'active') {
                // ✅ Filter: hanya kembalikan tugas yang isCompleted-nya FALSE
                filteredArray = tasks.filter(task => !task.isCompleted);
            } else if (filterType === 'completed') {
                // ✅ Filter: hanya kembalikan tugas yang isCompleted-nya TRUE
                filteredArray = tasks.filter(task => task.isCompleted);
            }
            
            // Render array hasil filter ke UI
            renderTasks(filteredArray);
            updateFilterButtonStyles(filterType);
        }

        // --- SETUP LISTENERS ---
        function setupDeleteListeners() {
            document.querySelectorAll('.delete-btn').forEach(button => {
                // Pastikan listener hanya dipasang sekali (penting jika ini diulang)
                // Di sini kita hapus listener lama, lalu pasang yang baru (untuk simplifikasi demo)
                button.removeEventListener('click', handleDeleteClick);
                button.addEventListener('click', handleDeleteClick);
            });
        }

        function handleDeleteClick(event) {
            // Dapatkan ID dari data-id tombol yang diklik
            const taskId = parseInt(event.target.getAttribute('data-id'));
            deleteTask(taskId);
        }

        function setupFilterButtons() {
            document.querySelectorAll('.filter-btn').forEach(button => {
                button.addEventListener('click', (event) => {
                    const filterType = event.target.getAttribute('data-filter');
                    filterTasks(filterType);
                });
            });
        }
        
        function updateFilterButtonStyles(activeFilter) {
            document.querySelectorAll('.filter-btn').forEach(button => {
                const filterType = button.getAttribute('data-filter');
                if (filterType === activeFilter) {
                    button.classList.remove('bg-gray-200', 'text-gray-800');
                    button.classList.add('bg-indigo-500', 'text-white');
                } else {
                    button.classList.remove('bg-indigo-500', 'text-white');
                    button.classList.add('bg-gray-200', 'text-gray-800');
                }
            });
        }


        // --- INISIALISASI ---
        document.addEventListener('DOMContentLoaded', () => {
            setupFilterButtons();
            // Inisialisasi tampilan awal dengan filter 'all'
            filterTasks(currentFilter); 
        });
    </script>
</body>
</html>
