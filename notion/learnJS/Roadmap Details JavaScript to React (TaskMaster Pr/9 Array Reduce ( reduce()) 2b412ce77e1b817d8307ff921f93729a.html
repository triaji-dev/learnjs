9. Array Reduce (.reduce()) (15 Menit)

A. Konsep Dasar Array Method: .reduce()

Metode .reduce() adalah salah satu Higher-Order Function (fungsi tingkat tinggi) yang paling kuat dan serbaguna di JavaScript. Tujuannya adalah untuk mengambil seluruh array dan "mereduksinya" menjadi satu nilai tunggal.

Nilai tunggal hasil reduksi ini bisa berupa apa saja: angka, string, Boolean, objek, atau bahkan array lain.

Cara Kerja .reduce()

.reduce() bekerja dengan mengiterasi melalui setiap elemen array dan menjalankan fungsi callback di setiap iterasi. Fungsi callback ini memiliki parameter unik yang sangat penting: Akumulator.

Akumulator (accumulator):

  • Ini adalah nilai yang dibawa (diakumulasikan) dari satu iterasi ke iterasi berikutnya.
  • Pada akhir iterasi, nilai akhir dari akumulator akan menjadi nilai return dari keseluruhan metode .reduce().

Sintaks Dasar:

JavaScript

array.reduce((accumulator, currentValue, index, array) => {
    // Logika perhitungan ada di sini
    return nextAccumulatorValue; // Mengembalikan nilai baru untuk akumulator
}, initialValue); // <--- Nilai Awal (Initial Value) untuk Akumulator
  • accumulator (atau acc): Nilai dari hasil akumulasi sebelumnya.
  • currentValue (atau curr): Elemen array yang sedang diproses saat ini.
  • initialValue (Opsional): Nilai awal yang akan digunakan akumulator pada iterasi pertama. Sangat dianjurkan untuk selalu menyertakan ini untuk mencegah ambiguitas tipe data.

B. Implementasi di Fitur "TaskMaster" (Widget Statistik)

Dalam fitur TaskMaster, kita sering kali perlu menghitung statistik dasbor, seperti total tugas selesai dan total tugas pending.

Tantangan: Bagaimana cara menghitung dua nilai berbeda (selesai dan pending) hanya dalam satu kali iterasi melalui array tugas?

Solusi: Menggunakan Objek sebagai Akumulator.

Kita dapat mengatur initial value akumulator sebagai objek yang menyimpan kedua hitungan tersebut.

Contoh Kasus: Menghitung total tugas selesai vs. pending.

JavaScript

const tasks = [
    { id: 1, title: 'Meeting Client', isCompleted: true },
    { id: 2, title: 'Kirim Email', isCompleted: false },
    { id: 3, title: 'Review Code', isCompleted: true },
    { id: 4, title: 'Riset Fitur Baru', isCompleted: false },
    { id: 5, title: 'Update Dokumen', isCompleted: true },
];

// Akumulator diinisialisasi sebagai objek
const taskStats = tasks.reduce((stats, task) => {
    if (task.isCompleted) {
        // Jika tugas selesai, tambahkan 1 ke properti 'completed'
        stats.completed += 1;
    } else {
        // Jika tugas pending, tambahkan 1 ke properti 'pending'
        stats.pending += 1;
    }
    
    // Sangat penting: kembalikan objek akumulator yang telah diperbarui
    return stats; 
}, { completed: 0, pending: 0 }); // Nilai Awal (Initial Value)

/*
Hasil (taskStats):
{ 
    completed: 3, 
    pending: 2 
}
*/

Keuntungan pendekatan ini adalah efisiensi. Kita menghindari melakukan dua loop terpisah (.filter() untuk selesai, lalu .filter() lagi untuk pending) dan menyelesaikannya dalam satu kali pass melalui data.

C. Relevansi React: Complex Logic / useMemo

Membuat State Turunan yang Kompleks

Di React, .reduce() sangat sering digunakan untuk menghitung Derived State (data turunan) yang lebih kompleks daripada sekadar string atau Boolean sederhana.

Optimasi dengan useMemo

Jika Anda menghitung statistik yang berat (misalnya, memproses ribuan data) dan hasil perhitungannya hanya dibutuhkan untuk ditampilkan (seperti Widget Statistik), Anda harus membungkus logika .reduce() di dalam React Hook useMemo.

Tujuan useMemo:

  • Mencegah perhitungan ulang yang mahal dari fungsi reduce pada setiap render komponen.
  • Perhitungan hanya akan dilakukan ulang jika dependency arraynya (dalam kasus ini, array tasks) benar-benar berubah.

Contoh Pseudo-Code React:

JavaScript

import React, { useMemo } from 'react';

function DashboardWidget({ tasks }) {
    
    // Perhitungan kompleks dibungkus dalam useMemo
    const taskStats = useMemo(() => {
        console.log("Menghitung ulang statistik tugas...");
        return tasks.reduce((stats, task) => {
            // ... (Logika .reduce() seperti di atas)
            if (task.isCompleted) {
                stats.completed += 1;
            } else {
                stats.pending += 1;
            }
            return stats;
        }, { completed: 0, pending: 0 });
    }, [tasks]); // Dependensi: Hanya hitung ulang jika array 'tasks' berubah.

    return (
        <div className="stats">
            <p>Total Selesai: {taskStats.completed}</p>
            <p>Total Pending: {taskStats.pending}</p>
        </div>
    );
}

Live Coding: Menghitung Statistik Tugas (15 Menit)

Dalam sesi ini, kita akan menerapkan .reduce() untuk menghitung statistik tugas dan menampilkannya di widget dasbor.

Exercise untuk peserta:

  1. Deklarasikan array tugas dengan properti isCompleted.
  1. Buat fungsi calculateTaskStats(tasks) yang menggunakan .reduce() untuk menghasilkan objek { completed: N, pending: M }.
  1. Tulis fungsi renderDashboard() yang mengambil hasil dari calculateTaskStats dan menggunakan Template Literals untuk menyusun UI widget statistik.
  1. Panggil renderDashboard() untuk menampilkan hasilnya di HTML.

Contoh Code Starter (HTML/JS):

HTML

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Praktek: Array Reduce (.reduce())</title>

</head>
<body class="bg-gray-100 p-8">

    <div id="app" class="max-w-3xl mx-auto bg-white p-8 rounded-lg shadow-xl">
        <h1 class="text-3xl font-bold mb-8 text-gray-800 border-b pb-4">TaskMaster Dashboard</h1>
        
        <div id="stats-widget">
            <p class="text-center text-gray-500">Menghitung statistik...</p>
        </div>
        
        <h2 class="text-xl font-semibold mt-10 mb-4 text-gray-700">Daftar Tugas (Untuk Referensi)</h2>
        <ul id="task-list" class="space-y-2 text-sm text-gray-600">
            </ul>
    </div>

    <script>
        // --- DATA STATE APLIKASI ---
        const TASKS_DATA = [
            { id: 1, title: 'Siapkan Slide Kelas Online', isCompleted: true, priority: 'High' },
            { id: 2, title: 'Cek Email Pagi', isCompleted: true, priority: 'Low' },
            { id: 3, title: 'Perbaiki Bug di Modul Auth', isCompleted: false, priority: 'High' },
            { id: 4, title: 'Riset Metode .reduce()', isCompleted: false, priority: 'Medium' },
            { id: 5, title: 'Makan Siang', isCompleted: true, priority: 'Low' },
            { id: 6, title: 'Desain Layout Dashboard', isCompleted: false, priority: 'Medium' },
            { id: 7, title: 'Ulangi Materi .filter()', isCompleted: false, priority: 'Low' },
        ];

        const statsWidgetContainer = document.getElementById('stats-widget');
        const taskListContainer = document.getElementById('task-list');

        // --- FUNGSI UTAMA: MENGGUNAKAN .REDUCE() ---
        function calculateTaskStats(tasks) {
            console.log("Memulai perhitungan statistik menggunakan .reduce()");
            
            // ✅ Gunakan .reduce() dengan Objek sebagai Akumulator
            return tasks.reduce((stats, task) => {
                
                // Logika: Cek status isCompleted
                if (task.isCompleted) {
                    stats.completed += 1;
                } else {
                    stats.pending += 1;
                }
                
                // Juga, hitung total prioritas (Contoh kompleks lain)
                if (task.priority === 'High') {
                    stats.highPriority += 1;
                }
                
                return stats; // Kembalikan objek akumulator yang sudah diperbarui
            }, { 
                // Initial Value (Sangat penting!)
                completed: 0, 
                pending: 0,
                total: tasks.length,
                highPriority: 0
            });
        }

        // --- FUNGSI RENDER DASHBOARD ---
        function renderDashboard() {
            // 1. Hitung Statistik (statik turunan)
            const stats = calculateTaskStats(TASKS_DATA);
            
            // 2. Render Widget Statistik menggunakan Template Literals
            const widgetHTML = `
                <div class="grid grid-cols-4 gap-4 text-center">
                    <div class="p-4 bg-gray-100 rounded-lg border-l-4 border-gray-500">
                        <p class="text-sm font-semibold text-gray-500">Total Tugas</p>
                        <p class="text-3xl font-bold text-gray-900">${stats.total}</p>
                    </div>
                    <div class="p-4 bg-green-100 rounded-lg border-l-4 border-green-500">
                        <p class="text-sm font-semibold text-green-700">Selesai</p>
                        <p class="text-3xl font-bold text-green-900">${stats.completed}</p>
                    </div>
                    <div class="p-4 bg-red-100 rounded-lg border-l-4 border-red-500">
                        <p class="text-sm font-semibold text-red-700">Pending</p>
                        <p class="text-3xl font-bold text-red-900">${stats.pending}</p>
                    </div>
                     <div class="p-4 bg-yellow-100 rounded-lg border-l-4 border-yellow-500">
                        <p class="text-sm font-semibold text-yellow-700">Penting</p>
                        <p class="text-3xl font-bold text-yellow-900">${stats.highPriority}</p>
                    </div>
                </div>
            `;
            
            statsWidgetContainer.innerHTML = widgetHTML;
            
            // Opsional: Render Daftar Tugas
            taskListContainer.innerHTML = TASKS_DATA.map(task => {
                const status = task.isCompleted ? '✅' : '⏳';
                return `<li>${status} [${task.priority}] ${task.title}</li>`;
            }).join('');
        }

        // --- INISIALISASI ---
        document.addEventListener('DOMContentLoaded', renderDashboard);
    </script>
</body>
